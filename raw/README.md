# raw/

Your source documents live here. Drop files in and leave them alone — this folder is read-only for Claude. Nothing here is ever modified or deleted by the assistant.

## Subfolders

| Folder | What goes here |
|--------|---------------|
| `claude-exports/` | Conversation exports from Claude |
| `chatgpt-exports/` | Conversation exports from ChatGPT |
| `gemini-exports/` | Conversation exports from Gemini |
| `notion-exports/` | Pages and databases exported from Notion |
| `notes/` | Freeform notes from any source |
| `articles/` | Saved articles, essays, newsletters |
| `fathom/` | Meeting transcripts and summaries from Fathom |
| `youtube-transcripts/` | Transcripts pulled from YouTube videos |
| `pdfs/` | PDFs: books, papers, reports |
