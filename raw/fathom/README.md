# raw/fathom/

Meeting transcripts and AI summaries from [Fathom](https://fathom.video).

## How to export

1. Open a completed call in Fathom
2. Click **Export** → choose **Transcript** or **Summary** (markdown recommended)
3. Save the file here, named by date and meeting title, e.g. `2026-06-19-team-standup.md`
