# raw/articles/

Saved articles, essays, blog posts, and newsletters.

## Suggested workflow

- Use a read-later tool (Readwise Reader, Pocket, Instapaper) and export to markdown
- Or save directly: copy article text into a `.md` file named by date and slug, e.g. `2026-06-19-title-of-article.md`
- Include the source URL at the top of each file
