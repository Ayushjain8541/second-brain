# raw/youtube-transcripts/

Transcripts from YouTube videos.

## How to get transcripts

- **Browser:** Open a video → **···** below the player → **Show transcript** → copy/paste
- **CLI tool:** `yt-dlp --write-auto-sub --skip-download --sub-format vtt <url>` then convert VTT to text
- **Services:** [Tactiq](https://tactiq.io), [Glasp](https://glasp.co), or similar browser extensions

Name files by date and video title, e.g. `2026-06-19-video-title.md`, and include the YouTube URL at the top.
